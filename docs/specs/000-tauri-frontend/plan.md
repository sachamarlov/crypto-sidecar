## API client codegen

1. `scripts/export_openapi.py` (Python) -- instantiate the FastAPI
   app via `create_app(...)` and dump `app.openapi()` to
   `frontend/openapi.json`.
2. `pnpm openapi:generate` -- runs `openapi-typescript ../openapi.json
-o src/api/schema.d.ts`. Pinned to a version compatible with
   FastAPI's OpenAPI 3.1 output.
3. `src/api/client.ts` -- `createClient<paths>({ baseUrl, headers })`
   - interceptor injecting `X-GuardiaBox-Token` and (when present)
     `X-GuardiaBox-Session`.

## State management split (ADR-0017 candidate)

- **Jotai** for lock state -- atomic, fine-grained reactivity, every
  consumer subscribes to the precise atom they read.
- **Zustand** for UI globals (theme, language) -- one global store,
  no atom proliferation for non-reactive surfaces.
- **TanStack Query** for server state -- HTTP cache + invalidation +
  optimistic update.

## Auto-lock

A `useAutoLock()` hook ticks an interval against
`expiresAtAtom`. When `Date.now() > expiresAt`, it dispatches
`POST /api/v1/vault/lock` (best-effort) and resets the lock atom
-- AuthGuard reroutes to `/lock`.

## i18n (NFR-6)

- `i18next-browser-languagedetector` reads `localStorage.guardiabox.lang`
  -> `navigator.language` -> fallback `en`.
- `react-i18next` `<Trans>` for inline JSX.
- Key namespacing: `lock.title`, `dashboard.empty`, `encrypt.kdf.tooltip`.
- A pre-commit script (`scripts/check_i18n_keys.ts`, optional)
  diffs `t('...')` calls against `fr.json` ∪ `en.json`.

## Accessibility (NFR-7)

- shadcn primitives provide ARIA out of the box.
- Focus visible: Tailwind `outline-ring/50` on `:focus-visible`.
- Reduced motion: `useReducedMotion()` from Framer Motion + a CSS
  `@media (prefers-reduced-motion)` killing keyframe animations.
- E2E: `injectAxe(page); checkA11y(page, null, { detailedReport: true })`.

## Test plan

- **Vitest** unit tests on hooks + stores + components critiques.
- **Playwright** E2E on `lock`, `encrypt-decrypt round-trip`,
  `share-and-accept`, `anti-oracle byte-identical`, `auto-lock`,
  `audit-history`, `secure-delete`, `users-crud`.
- **axe-playwright** a11y audits on every screen.

## CI activation

Committing `pnpm-lock.yaml` triggers the `frontend` job in
`.github/workflows/ci.yml` (currently skipped). The job runs
`pnpm install --frozen-lockfile && pnpm lint && pnpm typecheck &&
pnpm test --run`. Adding `pnpm test:e2e` is gated on a Playwright
browser cache to keep the runner fast.
